# Roadmap

## V1 now
- landing page
- no-login builder flow
- support for 4 appeal types
- case input form
- evidence file upload
- review page
- PDF export
- no database
- no persistent storage

## Later
- improved PDF layout
- ZIP export with attachments
- saved cases
- user accounts
- better timeline generation
- Google-specific export improvements
- additional appeal types

## Not now
- team collaboration
- admin dashboard
- analytics
- direct Google API integration
- CRM/helpdesk integrations
- legal advice features
- fraud prevention system
- multi-platform support
